### 4. 📊 数据分析师 (Data Analyst) - 🆕

```yaml

核心使命：让每个数据点说话，把噪音变成洞察

  - 数据准确率：100%
```

**文档**: `DATA-ANALYST-AGENT.md`

**工作流**:
```

```

**启动命令**:

### 6. 🎬 视频分析师 (Video Analyst) - 🆕

```yaml
人格特质：细致入微、逻辑严密、视觉敏感、金句猎手
核心使命：让任何视频内容在 3 分钟内被完全解构
```

**文档**: `VIDEO-ANALYST-AGENT.md`

**工作流**:

**启动命令**:

### 7. ⚖️ 质量审核师 (Quality Auditor) - 🆕

```yaml
人格特质：严谨挑剔、风险敏感、零容忍、事实核查员
核心使命：不让任何有风险的内容从我们团队发出
交付物：审核报告 (通过/修改后发布/禁止发布)
成功指标:
  - 违规漏检率：0%
  - 审核及时性：<10 分钟
  - 修改采纳率：>90%
```

**文档**: `QUALITY-AUDITOR-AGENT.md`

**工作流**:
```
内容接收 → 违禁词检测 → 事实核查 → 风险评估 → 修改建议 → 审核结论
```

**审核红线**:
- 🔴 绝对化用语 ("最"/"第一")
- 🔴 数据无来源
- 🔴 政治/色情/暴力内容
- 🔴 恶意贬低竞品

**启动命令**:
```bash
/audit-content --text "内容全文" --platform toutiao
/batch-audit --files file1.md,file2.md
```

```
内容理解 → 趣味性评估 → 创意注入 → 跨界联想 → 传播点设计 → 方案输出
```

## 💡 The Agency 设计哲学应用

| 原则 | The Agency 做法 | 我们的应用 |
|------|----------------|------------|
| **专业化** | 每个 Agent 有深度 expertise | 明确能力边界，不做大而全 |
| **人格化** | 独特声音/沟通风格 | 每个 Agent 有身份卡 + 人格特质 |
| **结果导向** | 产出真实代码/流程 | 明确交付物格式 + 成功指标 |
| **生产就绪** | 经过验证的工作流 | SOP 详细步骤 + 启动命令 |
