# 封面设计师 Agent 方案

**版本**: 1.0  
**创建时间**: 2026-03-13 00:18  
**目标**: 自动生成小红书爆款封面

---

## 🎯 核心能力

### 1. 九宫格配图生成
```
输入：笔记内容
输出：9 张风格统一配图

功能:
- 自动提取关键词
- 匹配图片风格
- 保持色调一致
- 添加文字说明
```

### 2. 前后对比图
```
适用场景：
- 健身减肥
- 美妆护肤
- 家居改造
- 汽车改装

生成逻辑:
- Before 图片处理 (灰度/暗化)
- After 图片处理 (鲜亮/突出)
- 添加对比标签
```

### 3. 纯文字封面
```
适用场景:
- 干货总结
- 清单列表
- 公式方法

设计要素:
- 大字号标题
- 醒目配色
- 简洁布局
- Emoji 点缀
```

### 4. 风格切换
```
支持风格:
- 简约时尚
- 可爱手绘
- 商务专业
- 清新文艺
- 科技未来
```

## 📋 封面设计要素

### 尺寸规范
```
推荐比例：1:1 (1080x1080)
最小尺寸：600x600
最大尺寸：2000x2000
格式：PNG/JPG
```

### 文字规范
```
标题字号：60-80px
副标题号：40-50px
正文字号：30-40px
字体：思源黑体/站酷高端黑
```

### 配色规范
```
主色调：品牌色/主题色
辅助色：对比色/邻近色
文字色：黑白灰/高对比
背景色：纯色/渐变/纹理
```

### 布局规范
```
九宫格：3x3 网格
对比图：左右/上下分割
文字封面：居中/居左/居右
留白：10-20%
```
