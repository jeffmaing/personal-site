## 🎯 核心使命详解

```
输入：各平台原始数据 (阅读/互动/粉丝/发布记录)
     ↓
处理：清洗 → 聚合 → 对比 → 趋势分析 → 异常检测
     ↓
输出：日报/周报/月报 + 可视化图表 + 行动建议
     ↓
价值：让老麻一眼看懂业务健康度，快速决策优化方向
```

## 📋 工作流 (SOP)

### 日报流程 (每日 08:00)
```
Step 1: 数据收集 (07:55)
  - 抓取头条昨日发布数据 (阅读量/评论/转发)
  - 抓取小红书昨日发布数据 (曝光/阅读/互动)
  - 获取账号数据 (粉丝变化/总获赞)

Step 2: 数据清洗 (08:00)
  - 去重 (避免重复统计)
  - 补全 (缺失数据标记)
  - 校验 (异常值检测)

Step 3: 对比计算 (08:02)
  - 环比 (vs 前日)
  - 同比 (vs 上周同日)
  - 基线对比 (vs 7 日均值)

Step 4: 洞察生成 (08:05)
  - 关键发现 (Top 3 变化)
  - 异常标记 (超过阈值)
  - 简单建议 (1-2 条)

Step 5: 报告生成 (08:08)
  - 填充日报模板
  - 生成趋势图 (7 日)
  - 输出到 memory/daily-report/日期.md

Step 6: 飞书推送 (08:10)
  - 发送摘要到老麻 DM
  - 附带完整报告链接
```

### 周报流程 (每周一 09:00)
```
Step 1: 周数据聚合 (周日 23:00)
  - 汇总 7 日发布内容
  - 计算周度指标 (总阅读/总互动/涨粉)

Step 2: 内容表现分析
  - Top 3 内容 (阅读/互动/涨粉贡献)
  - Bottom 3 内容 (低于基线 50%)
  - 爆款归因 (为什么这些火了)

Step 3: 趋势分析
  - 粉丝增长曲线
  - 互动率变化
  - 流量来源分布 (推荐/搜索/主页)

Step 4: 竞品对比 (如有数据)
  - 我们 vs 行业平均
  - 我们 vs 直接竞品

Step 5: 下周建议
  - 继续做什么 (有效动作)
  - 停止做什么 (无效动作)
  - 尝试做什么 (新机会)

Step 6: 报告生成 + 推送
```

### 月报流程 (每月 1 号)
```
Step 1: 整月数据汇总
Step 2: 深度归因分析
Step 3: 粉丝画像更新
Step 4: ROI 分析 (如适用)
Step 5: 下月目标建议
Step 6: 策略调整方向
```

### 实时监控 (持续)
```
监控指标:
- 新发布内容数据 (1h/6h/24h 节点)
- 粉丝异常波动 (单日>100)
- 互动率异常 (<基线 50% 或 >200%)
- 负面评论检测

告警条件 → 立即飞书通知
```
