**版本**: 1.0

---

## ⚠️ 重要声明

**本 Agent 定位**: 辅助决策工具  
**不是**: 代客理财/投资建议  
**责任**: 用户自行承担投资风险

---

## 🎯 核心能力

### 1. 数据采集层
```
✅ 实时行情 (Crawlee + 新浪财经 API)
✅ 公司公告 (Crawlee 抓取)
✅ 行业新闻 (Crawlee 抓取)
✅ 研报摘要 (Crawlee 抓取)
✅ 社交媒体情绪 (小红书/微博/股吧)
```

### 2. 分析层
```
✅ 技术指标分析 (MA/MACD/RSI/KDJ)
✅ 基本面分析 (PE/PB/ROE/营收增长)
✅ 情绪分析 (正面/负面/中性)
✅ 资金流向 (北向资金/主力进出)
✅ 板块轮动 (热点追踪)
```

### 3. 决策层
```
✅ 买入信号 (多因子共振)
✅ 卖出信号 (止盈/止损)
✅ 仓位建议 (根据风险等级)
✅ 风险提示 (黑天鹅事件)
```

### 4. 执行层 (需人工确认)
```
❌ 自动交易 (合规风险)
✅ 交易提醒 (飞书推送)
✅ 条件单设置 (券商 APP)
✅ 盘后复盘 (自动生成报告)
```

## 📋 每日工作流

### 盘前 (09:00-09:30)
```
1. 隔夜外盘回顾
2. 重要新闻汇总
3. 持仓公司公告
4. 今日关注事项
```

### 盘中 (09:30-15:00)
```
1. 涨跌幅告警 (>5%)
2. 成交量异常提醒
3. 技术指标信号
4. 大单异动监控
```

### 盘后 (15:00-18:00)
```
1. 当日盈亏汇总
2. 龙虎榜分析
3. 资金流向统计
4. 明日策略建议
```

### 周末 (周六/日)
```
1. 周度复盘报告
2. 持仓股基本面更新
3. 下周关注事项
4. 调仓建议
```
