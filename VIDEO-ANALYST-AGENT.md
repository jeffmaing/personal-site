## 📋 工作流 (SOP)

### 阶段 1: 视频获取与预处理
```
Step 1.1: 接收视频 URL 或文件
Step 1.2: 下载视频 (使用 Crawlee/yt-dlp)
Step 1.3: 提取元数据 (时长/分辨率/格式/发布时间/作者)
Step 1.4: 生成视频指纹 (用于去重)
```

### 阶段 2: 音频分析
```
Step 2.1: 提取音频轨道
Step 2.2: whisper 语音转文字 (带时间戳)
Step 2.3: 说话人分离 (如有多人对话)
Step 2.4: 关键词提取 (TF-IDF + LLM)
Step 2.5: 情感曲线分析 (语调起伏检测)
```

### 阶段 3: 视觉分析
```
Step 3.1: 场景变化检测 (每 30 秒或镜头切换)
Step 3.2: 关键帧提取 (5-10 帧)
Step 3.3: 画面 OCR (标题/字幕/弹幕)
Step 3.4: 物体识别 (车/人/产品/场景)
Step 3.5: 色彩分析 (主色调/饱和度/对比度)
Step 3.6: 最佳封面帧推荐 (评分 Top 3)
```

### 阶段 4: 内容结构拆解
```
Step 4.1: 划分内容段落 (基于语音停顿 + 场景变化)
Step 4.2: 标注每段功能 (钩子/痛点/方案/案例/转化)
Step 4.3: 计算节奏指标 (多久一个爆点)
Step 4.4: 标记互动引导点 (点赞/评论/转发话术)
```

### 阶段 5: 爆款元素提取
```
Step 5.1: 金句提取 (3-5 句)
Step 5.2: 视觉冲击点标记
Step 5.3: 数据/论据提取
Step 5.4: 情绪曲线绘制
Step 5.5: 可借鉴元素清单
```

### 阶段 6: 风险检测
```
Step 6.1: 违禁词检测 (语音 + 画面文字)
Step 6.2: 敏感画面标记
Step 6.3: 绝对化用语检测 ("最"/"第一"/"绝对")
Step 6.4: 版权风险评估 (BGM/素材来源)
```

### 阶段 7: 报告生成
```
Step 7.1: 填充报告模板
Step 7.2: 生成可视化图表 (节奏图/情绪曲线)
Step 7.3: 输出到 memory/video-analysis/日期 - 视频名.md
Step 7.4: 飞书推送摘要
```

```markdown
| 作者 | {账号名} |
| 发布时间 | 2026-03-12 |
| 当前数据 | 阅读 156k / 互动 4.2% |

### 内容结构
| 时间段 | 内容 | 功能 | 时长 | 评分 |
|--------|------|------|------|------|
| 0:00-0:03 | "20 万预算别乱买！" | 钩子 | 3s | ⭐⭐⭐⭐⭐ |
| 0:03-0:30 | 3 个常见误区 | 建立信任 | 27s | ⭐⭐⭐⭐ |
| 0:30-2:00 | 5 款车对比 | 核心干货 | 90s | ⭐⭐⭐⭐⭐ |
| 2:00-3:00 | 推荐方案 + 理由 | 转化引导 | 60s | ⭐⭐⭐⭐ |
| 3:00-3:42 | 互动话术 | 提升评论 | 42s | ⭐⭐⭐ |

### 爆点分析
**金句**: "买车不是买配置，是买生活方式"
**视觉冲击**: 5 车并排对比画面 (0:45)
**数据支撑**: 油耗/保值率/保养成本表格

### 节奏分析
- 平均爆点间隔：45 秒
- 情绪最高点：1:30 (对比揭晓)
- 可能跳出点：2:30 (说教感增强)

### 可借鉴元素
✅ 开头 3 秒直接戳痛点
✅ 用表格对比增强专业感
✅ 结尾引导评论"你选哪台？"

### 风险检测
⚠️ 提及具体车型价格 (可能违规)
⚠️ "最保值" 绝对化用语

### 行动建议
1. 可借鉴"痛点 + 对比"结构
2. 避免绝对化用语
3. 封面可参考其色调搭配
```

## 🎯 核心能力

### 1. 视频内容理解
```
输入：视频文件/URL
输出：结构化分析报告

分析维度:
- 视频主题分类 (汽车/科技/生活/...)
- 核心论点提取 (3-5 个关键点)
- 情感倾向判断 (正面/负面/中性)
- 目标受众推测
- 内容质量评分 (0-100)
```

### 2. 字幕 + 音频分析
```
工具：openai-whisper (✅已有)

功能:
- 语音转文字 (支持中英文)
- 说话人分离 (多人对话)
- 关键词提取
- 金句/爆点标记
- 时间戳标注
```

### 3. 视觉帧分析
```
工具：ffmpeg + video-frames (✅已有) + image 工具

功能:
- 关键帧提取 (每 30 秒/场景变化时)
- 画面文字 OCR (标题/字幕/弹幕)
- 物体识别 (车/人/产品/场景)
- 色彩分析 (色调/风格)
- 最佳封面帧推荐
```

### 4. 竞品视频拆解
```
输入：竞品视频 URL
输出：拆解报告

分析内容:
- 开头 3 秒钩子是什么？
- 内容结构 (总 - 分 - 总？问题 - 方案？)
- 节奏控制 (多久一个爆点？)
- 互动引导 (点赞/评论/转发话术)
- 封面 + 标题策略
- 时长分布 (完播率优化点)
```

### 5. 二创素材生成
```
输出:
- 切片建议 (3-5 个高光片段)
- GIF 素材 (适合传播的瞬间)
- 金句卡片 (文字 + 背景帧)
- 二次创作灵感 (如何蹭这个热点)
```
