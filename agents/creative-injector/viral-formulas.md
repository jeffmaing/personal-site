## 公式 1: AI 动画爆款公式

```
【第一人称】+【时间承诺】+【AI 能力】+【明确产出】+【零门槛】

示例:
✅ "我要 1 分钟学会！AI 直出动画短片，有手就行"
✅ "我试了 10 个工具！AI 一键生成视频，3 分钟出片"
✅ "手残党福音！AI 自动做动画，零基础"

变量替换:
- 时间承诺：1 分钟/3 步/5 分钟/10 分钟
- AI 能力：AI 直出/AI 一键/AI 自动/AI 帮你
- 明确产出：动画短片/视频/头像/PPT/周报
- 零门槛：有手就行/零基础/手残党福音/无需基础
```

## 公式 2: 痛点焦虑公式

```
【数字具体】+【负面词】+【人群定位】+【解决方案】

示例:
✅ "这 5 个坑，90% 新手都踩过 (附避坑指南)"
✅ "花了 20 万买的教训，今天说给你听"
✅ "年轻人买车 5 大坑，每个都值钱"

变量替换:
- 数字具体：3 个/5 个/7 个/10 个
- 负面词：坑/教训/陷阱/套路/后悔
- 人群定位：新手/年轻人/第一次/小白
- 解决方案：避坑指南/血泪史/真相
```

---

## 公式 3: 反差预期公式

```
【别人都 XXX】+【我偏 XXX】+【理由/结果】

示例:
✅ "别人都推荐买新，我劝你看看二手 (省 5 万)"
✅ "都说电动车香，我为什么劝你三思"
✅ "20 万预算，我不推荐你买 SUV"

变量替换:
- 别人都：推荐/选择/认为/说
- 我偏：劝你/建议/不推荐/反对
- 理由/结果：省 X 万/真相/后悔/踩坑
```

---

## 公式 4: 数据支撑公式

```
【我统计/试了/对比】+【数字】+【发现/结论】

示例:
✅ "我统计了 1000 个车主，最后悔的不是买贵了"
✅ "我试了 7 款车，这 3 台最值得买"
✅ "对比了 50 款 SUV，这 3 个数据最重要"

变量替换:
- 动作：统计/试了/对比/测试/调研
- 数字：100/1000/7 款/50 个
- 发现：最后悔/最值得买/最重要/真相
```

---

## 公式 5: 故事叙事公式

```
【我有个朋友】+【经历】+【现在/结果】

示例:
✅ "我有个朋友，当年非要买 XX 车，现在..."
✅ "我有个朋友，花了 30 万买教训"
✅ "我有个朋友，用 AI 做动画月入 3 万"

变量替换:
- 经历：买了/做了/试了/花了
- 现在/结果：后悔了/成功了/月入 X 万
```

---

## 公式 6: 名人效应公式

```
【名人】+【说了/做了】+【数字冲击】+【悬念】

示例:
✅ "傅盛说了一句话，55 万人沉默了"
✅ "雷军内部讲话：3 万块的车才是真智能"
✅ "周鸿祎一句话，AI 圈炸锅了"

变量替换:
- 名人：傅盛/雷军/周鸿祎/马斯克
- 数字冲击：55 万人/全网/整个行业
- 悬念：沉默了/炸锅了/破防了
```
