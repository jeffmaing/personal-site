BMW 沙盘系统
把 Excel 搬上 Web，保留全部计算逻辑

项目概览

HTML 页面
44
提取公式
1,359
跨表引用
678
SVG 元素
85

挑战

Excel 有 44 个 Sheet，每个都有复杂的公式和跨表引用。要在 Web 上 100% 复刻，包括 S 曲线可视化和循环引用计算。这不是简单的"截图转 HTML"，而是要保留完整的计算能力。

解决方案

1. 从源文件解析，不硬编码颜色和数据
2. 构建完整依赖图，识别 25 个循环引用
3. 实现 JS 计算引擎，14 项指标 100% 对齐
4. 彩色 S 曲线从单元格边缘出发，用 SVG 实现

成果

✅ 44 个 Sheet → 44 个 HTML 页面
✅ 14 项指标 100% 对齐 Excel
✅ 35 家经销商 KPI 数据库构建完成
✅ 经验文档 296 行（可复用方法论）
