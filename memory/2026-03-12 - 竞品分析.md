### 1.2 爆款标题公式（汽车赛道）

```
爆款标题 = 数字冲击 (30%) + 情绪共鸣 (30%) + 利益相关 (25%) + 悬念疑问 (15%)
```

**高点击标题模板**:

1. **价格敏感型**: "XX 车突然降价 X 万，老车主坐不住了"
2. **技术对比型**: "比亚迪 vs 蔚来，闪充和换电到底谁赢了？"
3. **政策影响型**: "油价又要涨！3 月 13 日后加满一箱多花 XX 元"
4. **行业趋势型**: "2026 年价格战终结？这家车企率先涨价"
5. **用户背刺型**: "刚提车就亏 X 万，XX 车主维权现场"

### 1.3 高读完率内容结构

**五段式结构**（适合微头条/短文）:
```
1. 热点引入 (1-2 句) — "刚刚，XX 宣布..."
2. 核心信息 (2-3 句) — 关键数据/价格/变化
3. 影响分析 (2-3 句) — 对消费者/行业的影响
4. 观点输出 (1-2 句) — 个人判断/预测
5. 互动引导 (1 句) — "你怎么看？评论区聊聊"
```

**深度文结构**（适合长文）:
```
1. 悬念开头 — 抛出反常识观点/数据
2. 背景铺垫 — 事件来龙去脉
3. 多方分析 — 厂家/经销商/用户视角
4. 深度解读 — 行业趋势/技术对比
5. 结论建议 — 给消费者的购买建议
```

### 2.2 爆款封面公式

```
爆款封面 = 高清实车图 (60%) + 关键信息文字 (30%) + 情绪表达 (10%)
```

**封面文字技巧**:
- 价格信息: "落地 15 万""月付 2000"
- 对比冲击: "油车 vs 电车 3 年账单"
- 情绪表达: "后悔没早买""真香警告"
- 数字冲击: "3 个缺点 5 个优点"

### 2.3 高互动文案结构

**AIDA 模型应用**:
```
Attention: "提车 3 个月，说点大实话..."
Interest: "当初纠结油电，现在算笔账"
Desire: "每月省 XXX，真金白银"
Action: "想看详细账单的评论区扣 1"
```

**五步分享法**:
```
1. 我的需求 (通勤/家庭/预算)
2. 我的纠结 (油车 vs 电车 vs 混动)
3. 我的选择 (最终选定 XX 的理由)
4. 我的体验 (3 个月真实感受)
5. 我的建议 (给纠结的人)
```

## 三、双平台可复用方法论

### 3.1 平台差异对比

| 维度 | 今日头条 | 小红书 |
|------|----------|--------|
| 用户心态 | 看新闻/获取信息 | 找攻略/看真实体验 |
| 内容偏好 | 热点/数据/观点 | 真实/细节/可参考 |
| 标题风格 | 信息密度高/冲击力强 | 情绪化/个人化/emoji |
| 互动逻辑 | 评论讨论观点 | 收藏备用 + 私信咨询 |
| 变现路径 | 流量分成/专栏 | 种草带货/线索转化 |

### 3.2 通用爆款公式

```
爆款 = 吸睛标题 (30%) + 价值内容 (40%) + 互动设计 (15%) + 时机选择 (15%)
```

**标题技巧 (双平台通用)**:
- 用数字: "3 个缺点""5 年账单""省了 2 万"
- 用情绪: "后悔""真香""没想到"
- 用痛点: "还在纠结油电？""选车避坑"
- 用对比: "油车 vs 电车""比亚迪 vs 特斯拉"

**内容价值 (双平台通用)**:
- 实用价值: 购车建议/成本计算/避坑指南
- 情感价值: 车主共鸣/选择认同/决策支持
- 社交价值: 谈资/观点表达/圈层归属

### 3.3 风险预警

| 风险 | 头条 | 小红书 |
|------|------|--------|
| AI 内容 | 严格打击，标记后限流 | 逐步加强治理 |
| 价格信息 | 需准确，虚假宣传违规 | 需真实，避免误导 |
| 品牌负面 | 客观陈述，避免造谣 | 真实体验，可批评但需有据 |
| 跨平台搬运 | 查重机制严格 | 原创度要求高 |
