### 2. 发布流程集成 (`src/post.js`)

**修改点**:
- 在标题/正文/上传/发布 4 个关键步骤添加自愈逻辑
- 失败时自动触发自愈流程
- 自愈成功后使用新选择器重试
- 自愈失败时抛出详细错误信息

**自愈流程**:
```
失败 → 截图 → 分析 → 尝试新选择器 → 验证 → 记录
          ↓
      成功 → 继续发布
          ↓
      失败 → 重试 (最多 3 次)
          ↓
      仍失败 → 通知人工介入
```
