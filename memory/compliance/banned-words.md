## 🔴 广告法违禁词 (绝对禁止)

```
最、第一、绝对、永久、独家、国家级、世界级、顶级、
首选、领先、开创、革命性、填补国内空白、100%、
完全、彻底、万能、根治、特效
```

**替换建议**:
- "最保值" → "保值率较高"
- "第一" → "领先"
- "100%" → "绝大多数"

---

## 🟠 金融敏感词 (需谨慎)

```
保本、无风险、稳赚、高收益、投资回报、
理财推荐、买入建议、目标价
```

**使用说明**: 涉及金融建议需加免责声明

---

## 🟡 医疗敏感词 (避免)

```
治疗、治愈、疗效、偏方、秘方、医生推荐、
处方、药方、康复
```
