Before / After · 范式转变

从手动到智能

Before · 旧模式

找资料 → 理逻辑 → 写内容 → 排版 → 美化 → 改到崩溃

  • 打开 PPT 软件，面对空白页发呆
  • 从模板库里挑一个"还凑合"的
  • 复制粘贴，逐页调整字号和对齐
  • 导出前发现第三页的标题溢出了
After · Agent 模式

丢内容 → 出一版 70 分 PPT → 微调

  • Agent 自动拆逻辑、分页面节奏
  • 设计系统已内置，不用挑模板
  • 信息层级、视觉节奏自动规划
  • 改 inline style 就行，3 分钟完事
